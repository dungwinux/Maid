# Maid
Package manager

Status: **Alpha**

Maid is a binary package manager, written in Python 3.7, for Windows.


### System Requirement

- Visual C++ 2017


### WIP Features

- [x] Environment Setup
- [x] Command-line interface
- [ ] Import/Export package list
    - [ ] Binding package
- [ ] Read MSI package

See [here](/guide.md) for usage guide.
